About Course

This Course provides an introductory technical overview of VMware Tanzu Kubernetes Grid including all editions.

Requirements

Prior to taking this course, please complete the Introduction to VMware Tanzu Portfolio Course.

You should also have a foundational understanding of Kubernetes before taking this course. If you are not already familiar with Kubernetes architecture, you can find some links to resources where you can learn Kubernetes basics:

  • Free online Kubernetes and cloud native technology training from KubeAcademy by VMware
  • Free online Introduction to Kubernetes course from The Linux Foundation
  • VMware Modern Application Solutions Gitbook
  • Free Kubernetes and Cloud Native Technology eBooks from the Tanzu Content Library
